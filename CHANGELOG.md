## v1.0.0 (2026-07-01) — 初始版本

### ✨ 核心功能
- 菜单栏常驻显示网络状态信号格（0-4 格）
- 并发探测国内外主流网站的响应时间
- 根据平均延迟自动评估网络质量等级
- 下拉菜单实时展示各网站精确延迟数据

### 🎨 界面优化
- **图标尺寸**：状态栏信号格缩小至 90%（22px → 20px），更紧凑协调
- **网站设置页重构**：分层布局——标题行 + 添加区 + 分隔线 + 配置列表
- **表格列设计**：名称 (80px)、地址 (120px)、类型、启用、操作五列，总宽 360px
- **删除按钮**：红色 ✕ 图标（18×18px），点击需二次确认弹窗
- **单元格对齐**：名称/地址左对齐垂直居中，其余列完全居中
- **自动隐藏滚动条**：平滑滚动体验

### 🔧 技术改进
- **测速协议升级**：HEAD → GET，解决小红书/X 等网站 HEAD 不兼容问题
- **偏好持久化**：UserDefaults JSON 存储，支持自定义监测网站增删改查
- **开机自启**：LaunchAgent plist 管理
- **自动更新**：GitHub Releases API 对比检查
- **零第三方依赖**：纯 Swift AppKit，SPM 构建
